CCM Reinstall Tool - Changelog
================================

v1.2.0
------
- The tool now checks for updates on every launch and can update itself
  automatically. It checks a version file online, and if a newer version
  is available it shows the changelog and either updates right away or
  asks first, depending on how important the update is
- Added a standalone updater that downloads new files, backs up whatever
  it's replacing first, then relaunches the tool - nothing is ever
  deleted or lost in the process
- Script now runs from CurrentVersion\ instead of the SoftwareCenter root
- Install arguments moved out of the script into Config\CmdLine.txt
  instead of being hardcoded
- Added a file presence check on launch - flags missing i386\, x64\,
  ccmsetup.cab, or CMClientInstall.wse before doing anything else
- sc.exe delete recovery logic deduplicated into one helper function
- Shows "up to date" on launch when no update is available
